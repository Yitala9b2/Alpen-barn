# Alpen-barn
Сайт для компании по строительству домов в стиле барн-хаус на галп, в работе использовались js, html, scss 
<div  id="ladesign_description" align="center" >
  
  <p> В данной работе использовались: слайдеры: swiper, попапы, бургер, прокрутка к нужному блоку, фильтр isotope.js, анимации: animate.css, формат картинок подстраивается под браузер, выбирая наиболее подходящий формат, адаптив и многое многое другое. Сайт собирал Webpack,Gulp. Был создан кастомный калькулятор с последующей  отправкой формы</p>
